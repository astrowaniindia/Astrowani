# EnableX Room Management

Rooms are the foundational units of communication in EnableX.

## Room Creation (REST API)
Rooms are typically created by your backend server via the EnableX Video API.

- **Endpoint:** `POST https://api.enablex.io/video/v2/rooms`
- **Authentication:** Basic Auth (`AppID:AppKey`)
- **Key Parameters:**
    - `name`: (Optional) Custom name for the room.
    - `owner_ref`: Your internal ID for the room creator.
    - `settings`: Configuration for participants, quality, and features.
        - `participants`: Max number of people allowed.
        - `mode`: `group` (SFU), `p2p` (Direct), or `lecture`.
        - `scheduled`: Whether it's a timed session.
        - `adhoc`: If true, the room is deleted after the last participant leaves.

## Room Joining (Client SDK)
Clients join rooms using the JWT token generated by your server.

- **SDK Method:** `room.connect(token)`
- **Prerequisites:**
    - A valid token.
    - Media permissions (Camera/Mic) granted by the user.
    - Local stream initialized (recommended before connecting).

## Room Leaving
Participants can leave a room voluntarily or be disconnected.

- **SDK Method:** `room.disconnect()`
- **Impact:**
    - Triggers `user-disconnected` event for all other participants.
    - If the room is `adhoc` and this was the last participant, the room is destroyed on the server.

## Participant Lifecycle
1. **Connecting:** The SDK is establishing a signaling link.
2. **Connected:** The user is officially in the room and can receive remote streams.
3. **Publishing:** The user is sending their own media (audio/video) to others.
4. **Subscribed:** The user is receiving media from another participant.
5. **Disconnected:** The session has ended for this user.

### Moderator Controls
If a participant joins with a "Moderator" token, they gain extra capabilities:
- **Kick Participant:** Remove someone from the room.
- **Mute All:** Force mute audio/video for everyone else.
- **Start/Stop Recording:** Control server-side session recording.
- **Lock Room:** Prevent new participants from joining.
